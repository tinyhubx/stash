# 리소스 저장소

> 이미지, 아이콘 등 다양한 리소스를 모아 관리하고 쉽게 사용할 수 있는 저장소입니다

---

## 📂 구조

```
stash/
├── avatars/     # 아바타
├── icons/       # 아이콘
├── images/      # 이미지
├── unsplash/    # Unsplash 무료 이미지
├── misc/        # 기타 자료
└── README.md
```

---

## 🌟 사용 방법

1. GitHub에서 폴더를 직접 탐색하여 자료를 확인합니다.
2. jsDelivr CDN을 통해 프로젝트에서 직접 사용할 수 있습니다 (권장):

```html
<img src="https://cdn.jsdelivr.net/gh/tinyhubx/stash/avatars/avatar1.png" alt="avatar">
```

```css
background-image: url('https://cdn.jsdelivr.net/gh/tinyhubx/stash/images/bg1.jpg');
```

```html
<img src="https://cdn.jsdelivr.net/gh/tinyhubx/stash/icons/icon1.svg" alt="icon">
```

---

## ⚠️ 저작권 및 안내

> 본 저장소는 학습 및 참고용입니다. 사용 책임은 사용자에게 있습니다.

### 1. 인터넷 수집 자료 (avatars/, icons/, images/, misc/)

- 대부분 인터넷 공개 자료에서 수집됨
- 출처 및 원작자가 불명확할 수 있음
- 저작권이 존재할 수 있음
- 본 저장소는 어떠한 사용 권한도 제공하지 않음

권장 사용:
- 개인 프로젝트
- 학습
- 비상업적 용도

```html
<!-- 출처: 인터넷 수집 -->
```

---

### 2. Unsplash 자료 (unsplash/)

- 출처: https://unsplash.com/
- 라이선스: https://unsplash.com/license

요약:
- 개인/상업적 사용 가능
- 수정 가능
- 출처 표기 필수 아님 (권장)
- 금지:
  - 원본 이미지 재판매
  - 경쟁 서비스 제작

```html
<!-- 출처: Unsplash -->
```

---

### 3. 저장소 안내

- 별도의 통합 라이선스 없음
- 각 파일은 원본 라이선스를 따름
- CDN은 단순 전달 역할
- 금지:
  - 불법 사용
  - 저작권 침해
