# 素材资源仓库

> 一个方便浏览、管理和引用的头像、图片和小素材合集

---

## 📂 仓库结构

```
stash/
├── avatars/     # 头像素材
├── icons/       # 图标素材
├── images/      # 图片素材
├── unsplash/    # Unsplash 免费图片合集
├── misc/        # 零散小素材
└── README.md
```

---

## 🌟 使用说明

1. 在 GitHub 上直接浏览对应文件夹获取素材。
2. 可通过 jsDelivr CDN 直接在网页或项目中引用（推荐）：

**基础引用格式：**

```html
<img src="https://cdn.jsdelivr.net/gh/tinyhubx/stash/avatars/avatar1.png" alt="头像示例">
```

CSS 示例：

```css
background-image: url('https://cdn.jsdelivr.net/gh/tinyhubx/stash/images/bg1.jpg');
```

SVG 图标或其他文件示例：

```html
<img src="https://cdn.jsdelivr.net/gh/tinyhubx/stash/icons/icon1.svg" alt="图标示例">
```
