## 🌟 Usage

1. Browse folders directly on GitHub to find assets.
2. Use jsDelivr CDN for direct usage in your projects (recommended):

**Basic usage format:**

```html
<img src="https://cdn.jsdelivr.net/gh/tinyhubx/stash/avatars/avatar1.png" alt="avatar example">
```

CSS example:

```css
background-image: url('https://cdn.jsdelivr.net/gh/tinyhubx/stash/images/bg1.jpg');
```

SVG or other files:

```html
<img src="https://cdn.jsdelivr.net/gh/tinyhubx/stash/icons/icon1.svg" alt="icon example">
```
